Utilities for constructing team rosters from serialized data sources.

The utilities in this module translate plain dictionaries or JSON payloads into rich domain objects that the match engine understands. They are primarily used by the CLI entrypoint and test fixtures to quickly spin up realistic squads without hand-coding every player. The helper functions deliberately default to reasonable attribute values so that incomplete datasets remain usable while still producing valid Player and Team instances.
