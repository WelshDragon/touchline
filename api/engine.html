Engine Package

Core Modules

Core engine packages for simulation physics, roles, and orchestration.

High-level orchestration for real-time football match simulations.

The match engine glues together physics, role behaviour, and officiating to advance the simulation tick by tick. The RealTimeMatchEngine class exposes a control surface for running full matches, while MatchState captures mutable runtime context shared between subsystems.
